# LetsCodeSweater
Зроблено за відеоуроками: https://www.youtube.com/watch?v=jH17YkBTpI4&t=1s&ab_channel=letsCode
Використанно такі технології:
* Spring MVC
* Spring Security
* Spring Data
* Spring Data
* Thymeleaf
* Flyway

А також деякі, що пов'язані з фронтом:
* Bootstrap
* Turbolinks
* Recaptcha
